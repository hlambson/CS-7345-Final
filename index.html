The graph algorithms library is built in Rust then exported to WASM to be used as a JavaScript library.

Discover More
Breadth First Search

Find the path from a root node to a target via exploring all nodes at the current depth before moving to the next depth level of the graph

Depth First Search

Find the path from a root node to a target via exploring all nodes in a branch before backtracking and exploring the next branch

Kruskal's Minimum Spanning Tree

Discover the tree that includes every node in the graph that minimizes the sum of the weights of the edges of the graph

Why WASM?

WASM, or Web Assembly, allows for the library to be ran in JavaScript, which gives the advantage of being able to run the code in a browser. Instead of coding the library in JavaScript originally, coding it in Rust gives it a significant performance boost.

Multithreaded Capabilities

To increase the performance of the library further, each function has been given a separate, threaded version. This allows all of the functions to be run in parallel rather than sequentially.
